# Beam Reaction Calculator

A Python tool for calculating beam support reactions from applied point loads and positions.

## Features
- Calculates support reactions for a simply supported beam
- Handles multiple point loads
- Organized function-based structure
- Clear printed output

## Technologies Used
- Python
- Functions and loops
- Basic statics calculations

## How to Run
Run the script in Python:

python beam_reaction_calculator.py
